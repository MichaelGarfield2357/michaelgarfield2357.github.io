Physics Simulations

Description

Here are a few physics simulations I've made over the years. The pendulum simulations are done using the Verlet integration scheme. The n-body gravity simulation utilizes an algorithm called the Barnes-Hut algorithm that utilizes a data structure called an oct-tree. The basic intuition behind the algorithm is grouping far away objects as one object, while having high fidelity in nearby objects.
